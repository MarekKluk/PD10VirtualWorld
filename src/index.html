Virtual World

Virtual World is a turn-based game with different organisms. The map has 20x20 tiles and contains various animals, plants, and the player.

Rules:

  • In each turn, every organism performs an action in the order of initiative. The organisms with higher initiative move first. If organisms have equal initiative, the older organism has priority,
  • Every animal moves in a random direction each turn, if not specified otherwise,
  • If two animals of different species meet, they fight. The animal with higher strength wins,
  • If animals of the same species meet, they create a third animal in one of the free adjoining fields,
  • The plants do not move in any direction at each turn. Instead, they have a 10% chance of spreading to one of the adjoining fields if there is any unoccupied,
  • If an animal sets foot on a plant, it gets eaten,
  • You can add new organism, by clicking on an empty field,
  • Each organism has specific unique skills, that are described when you try to add new one.

Controls:

Player can move in all eight directions, or stay in the same place for the turn.

You can control the player with a numpad, or if your keyboard is not equipped with it, then with the buttons on the right side of the board.
